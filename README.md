# AI-Evaluator

AI-Evaluator is an AI-powered answer sheet evaluation platform designed to automate the assessment process for educational institutions. The system enables teachers to upload answer keys and students to submit answer sheets, after which AI evaluates responses, assigns marks, and generates detailed feedback and performance analytics.

## Problem Statement

Manual evaluation of answer sheets is time-consuming, repetitive, and difficult to scale. AI-Evaluator streamlines this process by automatically analyzing student responses and providing consistent, data-driven assessments.

## Features

### Teacher Portal

* Upload and manage answer keys
* Manage subjects and exam terms
* View student submissions
* Review evaluation results
* Track overall performance

### Student Portal

* Upload answer sheets (Images/PDFs)
* Receive instant evaluation results
* View marks and percentages
* Access detailed feedback
* Track academic performance

### AI Evaluation Engine

* Automatic answer extraction
* Intelligent question-answer matching
* Concept-based evaluation
* Automated mark allocation
* Feedback generation

## Workflow

1. Teacher uploads answer keys.
2. Student uploads answer sheets.
3. The system extracts answers from uploaded files.
4. Answers are matched with the corresponding answer keys.
5. AI evaluates responses and calculates marks.
6. Results and analytics are generated automatically.
